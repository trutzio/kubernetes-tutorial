Voraussetzungen
===============

.. _voraussetzungen:

Linux
-----

#. Debian 13 Server zum Beispiel bei Hetzner mit `apt update && apt upgrade`
#. Umgang mit `ssh`, `ssh-agent`, `ssh-add`, `ssh-keygen`, `sudo`, `su`, `bash`
#. Umgang mit `systemctl` zum Verwalten von Diensten
#. Umgang mit `apt` zum Installieren von Software

Visual Studio Code
------------------

#. Installation auf dem lokalen Rechner https://code.visualstudio.com/download
#. Installation der Erweiterung "Database Client JDBC" mit der Id `cweijan.dbclient-jdbc`

Git
---

#. Installation von Git mit `apt install git`
#. Umgang mit `git clone`

Docker
------

#. Kostenloses Docker Account (wird für die Verwendung von Docker Scout benötigt), siehe dazu auch https://docs.docker.com/accounts/create-account/
